# (C# 코딩) 파일 비교 툴 File Compare

-1줄소개: 양쪽의 파일을 받아서 비교하는 프로그램

-사용한플랫폼: 
	-C#, .NET Windows Forms, Visual Studio, GitHub

-사용한컨트롤:
	-Label, TextBox, ListBox, Button

-사용한기술과구현한기능:
	-Visual Studio를이용하여UI 디자인
    -string 클래스를이용한사용자입력데이터처리
